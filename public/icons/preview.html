📋 使用方式

1. 在 worksheet HTML 中直接複製對應的 <svg>...</svg> 區塊

2. 控制大小用 CSS width / height 或 style="width:48px"

3. 列印時完全清晰，不需要外部檔案

4. 所有 SVG 皆使用書蟲暖棕配色 (#8a5f33 / #c4925a / #d5b07c / #f3e8d5)
